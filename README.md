## 🛰 Enterprise Project Experience  
### Qualcomm – AWARE (IoT Device Management System)

Worked on an **enterprise-level Qualcomm AWARE platform** designed to manage and monitor **multiple intelligent device types** through web-based dashboards.

This platform handled **real-time device data, lifecycle management, and user-driven customization** across different applications.

### Device Types & Functionality:
- **Trackers**  
  Used for tracking parcels or assets moving from one location to another, providing **real-time location and sensor data** displayed on the UI.
  
- **Thin Trackers**  
  Supported **tilt sensing**, showing how much a package or object was tilted during transit.
  
- **Labels**  
  Detected events such as **box opening, light detection, or drop events**, and reported these events to the system.

Each device type had **distinct behavior, data structure, and UI representation**, all handled conditionally on the frontend.
